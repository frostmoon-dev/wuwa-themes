# Wuthering Waves Themes

A **fan‑made** collection of VS Code color themes inspired by the beloved characters of **Wuthering Waves**.  Each theme is pre‑configured for its character – simply select the theme you want.

## ❖ Features

- **10 character‑specific themes** - more to come!
- Light (`vs`) and dark (`vs-dark`) variants for every character.
- Simple installation via the VS Code Marketplace.
- Perfect for **WUWA fans** who want their editor to feel like the game world.

## ⤓ Installation

1. Open **Extensions** (`Ctrl+Shift+X`).
2. Search for **Wuthering Waves Themes** (publisher `frostmoon‑dev`).
3. Click **Install**.
4. Open the Command Palette (`Ctrl+Shift+P`) → **Preferences: Color Theme** → pick a character theme.

## ⚔ Ready‑to‑Use Themes

All themes are fully configured and ready to use. Just pick a character theme from the VS Code color‑theme picker – no further tweaks are required.
